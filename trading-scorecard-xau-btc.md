# Daily Setup Scorecard — XAUUSD & BTCUSD

(Imported from previous workspace on 2026-03-05; use as pre-trade checklist.)

Use before entry. Score each item 0/1.

## A) Macro Alignment (0–3)
- [ ] Macro direction supports bias
- [ ] No major upcoming event risk (or already priced and stable)
- [ ] Cross-asset confirmation (DXY/yields/risk sentiment) not contradictory

## B) Regime & Structure (0–3)
- [ ] Daily regime is clear (trend/range/transition)
- [ ] Structure supports bias (HH/HL for longs, LH/LL for shorts)
- [ ] Price location at meaningful level (S/R, demand/supply, breakout base)

## C) Trigger Quality (0–3)
- [ ] Clear trigger candle/structure break/retest
- [ ] Momentum confirmation (RSI context / expansion)
- [ ] Entry is not overextended from invalidation point

## D) Risk Quality (0–3)
- [ ] SL at logical invalidation + ATR sanity check
- [ ] RR >= 1:2 to TP2 (or strong management plan)
- [ ] Position size respects risk cap and correlation cap

## Grade Rules
- 10–12 = Grade A (full planned risk)
- 7–9 = Grade B (50% size)
- 0–6 = Grade C (skip)

## Mandatory Kill-Switch
- Stop trading for day if:
  - daily drawdown limit hit, OR
  - 2 consecutive impulsive/non-system trades, OR
  - major event volatility invalidates plan structure.
