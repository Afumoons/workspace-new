# Perps Execution Checklist (Scalp + Intraday)

Use this before **every** trade.

## A) Pre-Trade Gate (must be YES)
- [ ] I have clear higher-timeframe bias (15m/1h)
- [ ] Market structure is clear (trend or breakout), not chop
- [ ] Volume confirms direction
- [ ] Risk/Reward is at least 1:1.8
- [ ] No major macro event due in next 30–60 minutes
- [ ] I am calm (no FOMO / revenge / tilt)

If any box is NO → **NO TRADE**.

## B) Setup Type (choose one)
- [ ] Trend pullback + rejection confirmation candle
- [ ] Breakout → retest hold → confirmation close

## C) Position Plan (define before entry)
- Pair: __________
- Direction: Long / Short
- Entry: __________
- Stop loss (invalidation): __________
- TP1 (+1R): __________
- TP2 (+2R or structure target): __________
- Planned R multiple: __________

## D) Risk Limits (hard rules)
- [ ] Risk per trade = 1% (max 1.5%)
- [ ] Leverage cap respected (Scalp 5x–10x, Intraday 3x–5x)
- [ ] Only 1 open trade at a time
- [ ] Daily stop = 3R loss or 3 losing trades
- [ ] Weekly stop = 8R drawdown

## E) In-Trade Management
- [ ] SL is not moved wider
- [ ] Take partial at +1R
- [ ] Move SL to near breakeven after partial
- [ ] Exit remaining at +2R or structure break
